About the program

The program does 3 things:

  1. HD H264 file
  2. Thumbnail from the first and third second of the 4k file
  3. Multibit rate HLS files

The file is hardcoded on the api (found in directory Samples of Api) for now but logic is implemented (80%) for use of url-s to be sent to the server to process videos from those url-s.
